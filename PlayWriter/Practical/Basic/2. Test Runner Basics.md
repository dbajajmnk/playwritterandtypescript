# 🎭 Playwright Test Runner Basics (Incremental)

---

## 1. WHAT

The **Playwright Test Runner** is the engine responsible for executing test cases.

It manages:

* test execution
* browser lifecycle
* parallel execution
* retries
* reporting

It sits between your **test files** and the **browser**.

High-level execution:

```text
Test File → Test Runner → Browser → Actions → Assertions → Report
```

---

## 2. WHY

Without a test runner:

* tests must be executed manually
* no grouping of tests
* no parallel execution
* no retry mechanism
* no structured reporting

With Playwright Test Runner:

* tests run automatically
* suites can be grouped logically
* execution can be parallel
* failures can be retried
* reports are generated

---

## 3. WHEN TO USE

The test runner is used:

* every time you run `npx playwright test`
* when executing test suites
* when running tests in CI/CD pipelines
* when debugging test failures
* when running tests across browsers

---

## 4. HOW

Execution flow:

```text
Test Suite → Test Runner → Worker → Browser → Context → Page → Test Steps → Assertions → Report
```

SauceDemo example:

```text
Login Suite → Runner → Browser → Login → Validate Inventory → Report
```

---

## 5. REAL-LIFE ANALOGY

Think of the test runner as a **movie director**.

* Test file → Script
* Test runner → Director
* Browser → Actors
* Test steps → Scenes
* Report → Final movie

Without a director, scenes are not organized.
With a director, everything runs in a controlled sequence.

---

## 6. ENGINEERING VIEW

Core Test Runner components:

* **test()** → smallest executable unit
* **describe()** → group of tests
* **beforeEach()** → setup before every test
* **afterEach()** → cleanup after every test
* **fixtures** → injected dependencies (page, browser)
* **workers** → parallel execution units
* **reporter** → output generator

## File: `tests/auth/login.spec.ts`

```ts
import { test } from '@playwright/test';
import { LoginPage } from '../../pages/LoginPage';
import { InventoryPage } from '../../pages/InventoryPage';
import { USERS } from '../../test-data/users';

test.describe('Authentication - Login', () => {
  let loginPage: LoginPage;
  let inventoryPage: InventoryPage;

  test.beforeEach(async ({ page }) => {
    loginPage = new LoginPage(page);
    inventoryPage = new InventoryPage(page);

    await loginPage.goto();
    await loginPage.validateLoginPageVisible();
  });

  test('valid login should open inventory page', async () => {
    await loginPage.login(USERS.standard.username, USERS.standard.password);
    await inventoryPage.validateInventoryPageLoaded();
  });

  test('locked user should not login successfully', async () => {
    await loginPage.login(USERS.locked.username, USERS.locked.password);
    await loginPage.validateLoginErrorVisible();
  });
});
```

## File: `tests/checkout/checkout.spec.ts`

```ts
import { test } from '@playwright/test';
import { LoginPage } from '../../pages/LoginPage';
import { InventoryPage } from '../../pages/InventoryPage';
import { CartPage } from '../../pages/CartPage';
import { CheckoutPage } from '../../pages/CheckoutPage';
import { USERS } from '../../test-data/users';

test.describe('Checkout', () => {
  let loginPage: LoginPage;
  let inventoryPage: InventoryPage;
  let cartPage: CartPage;
  let checkoutPage: CheckoutPage;

  test.beforeEach(async ({ page }) => {
    loginPage = new LoginPage(page);
    inventoryPage = new InventoryPage(page);
    cartPage = new CartPage(page);
    checkoutPage = new CheckoutPage(page);

    await loginPage.goto();
    await loginPage.login(USERS.standard.username, USERS.standard.password);
    await inventoryPage.validateInventoryPageLoaded();
  });

  test('user should complete checkout successfully', async () => {
    await inventoryPage.addBackpackToCart();
    await inventoryPage.openCart();

    await cartPage.validateCartHasItems();
    await cartPage.clickCheckout();

    await checkoutPage.fillCheckoutInfo('Deepak', 'Bajaj', '148033');
    await checkoutPage.continueCheckout();
    await checkoutPage.finishCheckout();
    await checkoutPage.validateOrderSuccess();
  });
});
```

## File: `playwright.config.ts`

import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  testDir: './tests',
  fullyParallel: true,
  retries: 1,
  timeout: 30000,
  expect: {
    timeout: 5000,
  },
  reporter: [['html'], ['list']],
  use: {
    headless: true,
    screenshot: 'only-on-failure',
    trace: 'on-first-retry',
    video: 'retain-on-failure',
  },
  projects: [
    {
      name: 'chromium',
      use: { ...devices['Desktop Chrome'] },
    },
    {
      name: 'firefox',
      use: { ...devices['Desktop Firefox'] },
    },
    {
