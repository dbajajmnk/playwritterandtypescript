# 📘 1-Month Automation Upskilling Roadmap (TOC)

## 👤 Target Learner Profile

* 8+ years Manual Testing background
* Basic Java + Selenium exposure
* Programming beginner (restarting after long gap)
* Time available: 1 hour daily
* Goal: Move toward Automation Engineer role

By the end of this roadmap you should be able to:

* Rebuild programming confidence

* Build a starter automation framework
* Gain exposure to Playwright, API testing and CI/CD
