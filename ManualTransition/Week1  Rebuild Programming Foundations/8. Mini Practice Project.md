# 💻 Hands-On Lab (Foundation Pattern)

## Use Case 1 Login Flow Simulation

```java id="bdkzby"
public class LoginProject {

static boolean login(String username,String password){

if(username.equals("admin") && password.equals("1234")){
return true;
}
else{
return false;
}

}

public static void main(String[] args){

String username="admin";
String password="1234";

boolean result=login(username,password);

if(result){
System.out.println("Login Success");
}
else{
System.out.println("Login Failed");
}

}

}
```

---

## Use Case 2 Validation Exercise

```java id="i24ltr"
public class LoginValidationProject {

static boolean validateTitle(String actual,String expected){
return actual.equals(expected);
}

public static void main(String[] args){

String actualTitle="Dashboard";
String expectedTitle="Dashboard";

if(validateTitle(actualTitle,expectedTitle)){
System.out.println("Validation Passed");
}
else{
System.out.println("Validation Failed");
}

}

}
```
