# Exercise 1: Basic Git Flow

## Goal

Practice the basic Git lifecycle.

## Steps

1. Open terminal in project root.
2. Run:

```bash
git init
git status
```

3. Stage all files:

```bash
git add .
```

4. Commit files:

```bash
git commit -m "Initial commit"
```

5. View history:

```bash
git log --oneline
```

## Expected Outcome

You should see one commit in Git history.
