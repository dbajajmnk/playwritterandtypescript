# 3.5 REST Assured Introduction

## 🎯 What

REST Assured is a Java library used to automate API testing.

Focus:

* API automation concepts
* Starter tests

It helps automate:

* request sending
* response validation
* status assertions
* JSON checks

Without API automation libraries:

* more manual validation
* repetitive code
* weak service automation

---

## ❓ Why This Matters in Automation

UI testing alone is not enough.

API automation helps:

* test faster than UI
* validate services directly
* catch defects early
* support backend-heavy systems

REST Assured is widely used in Java automation stacks.

---

## ⏰ When You Use REST Assured

Use when:

* automating APIs
* validating backend services
* creating regression API suites
* integrating API tests with UI automation

# Starter Tests

## Basic GET Test

```java id="j7p2rv"
given()
.when()
.get("/users")
.then()
.statusCode(200);
```

Validates response code.

---

## Response Body Validation

```java id="m3r8qd"
then()
.body("name",
equalTo("Deepak"));
```

Checks response data.

---

## POST Starter Test

```java id="w5n4tx"
given()
.body(payload)
.when()
.post("/users")
.then()
.statusCode(201);
```

Create and validate.
