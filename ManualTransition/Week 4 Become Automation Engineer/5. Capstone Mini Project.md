* UI automation flow

# 🎯 Goal

Build a beginner-friendly capstone project using:

* Selenium for UI automation
* REST Assured for API verification
* TestNG for execution and assertions

This project will help the learner create the project step by step on their own.

---

# 🧩 Project Scenario

Use the public practice website:

```text
https://automationexercise.com
```

Use these test values:

```text
Search keyword = top
Expected product = Blue Top
```

Capstone flow:

1. Open UI
2. Search product on UI
3. Verify product in UI result
4. Call API using same search keyword
5. Verify API response
6. Compare UI result with API result

# 💻 Step 5 Create Capstone Test Class

Create file:

```text
src/test/java/AutomationExerciseCapstoneTest.java
```

Paste this complete code:

```java
import io.restassured.RestAssured;
import io.restassured.response.Response;
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.testng.Assert;
import org.testng.annotations.AfterClass;
import org.testng.annotations.BeforeClass;
import org.testng.annotations.Test;

import java.time.Duration;
import java.util.List;

public class AutomationExerciseCapstoneTest {

    private WebDriver driver;

    private final String baseUrl = "https://automationexercise.com";
    private final String searchKeyword = "top";
    private final String expectedProduct = "Blue Top";

    private String uiMatchedProduct;

    @BeforeClass
    public void setup() {
        driver = new ChromeDriver();
        driver.manage().window().maximize();
        driver.manage().timeouts().implicitlyWait(Duration.ofSeconds(10));

        RestAssured.baseURI = baseUrl;
    }

    @Test(priority = 1)
    public void uiAutomationFlow() {
        driver.get(baseUrl);

        // Click Products menu
        driver.findElement(By.xpath("//a[contains(text(),'Products')]")).click();

        // Search product
        driver.findElement(By.id("search_product")).sendKeys(searchKeyword);
        driver.findElement(By.id("submit_search")).click();

        // Validate searched products section
        WebElement searchedProductsHeading =
                driver.findElement(By.xpath("//*[contains(text(),'Searched Products')]"));
        Assert.assertTrue(searchedProductsHeading.isDisplayed(), "Searched Products heading not visible");

        // Capture all product names
        List<WebElement> productNames =
                driver.findElements(By.xpath("//*[contains(@class,'productinfo')]//p"));

        Assert.assertTrue(productNames.size() > 0, "No products returned on UI search");

        boolean found = false;
        for (WebElement product : productNames) {
            String name = product.getText().trim();
            if (name.equalsIgnoreCase(expectedProduct)) {
                uiMatchedProduct = name;
                found = true;
                break;
            }
        }

        Assert.assertTrue(found, "Expected product not found on UI: " + expectedProduct);
        System.out.println("UI Matched Product: " + uiMatchedProduct);
    }

    @Test(priority = 2)
    public void apiVerificationFlow() {
        Response response =
                RestAssured
                        .given()
                        .contentType("application/x-www-form-urlencoded; charset=UTF-8")
                        .formParam("search_product", searchKeyword)
                        .when()
                        .post("/api/searchProduct");

        Assert.assertEquals(response.getStatusCode(), 200, "API status code mismatch");

        String responseBody = response.getBody().asString();
        Assert.assertNotNull(responseBody, "API response body is null");
        Assert.assertFalse(responseBody.isBlank(), "API response body is empty");

        Assert.assertTrue(responseBody.contains("products"), "API response does not contain products key");
        Assert.assertTrue(responseBody.toLowerCase().contains(expectedProduct.toLowerCase()),
                "Expected product not present in API response: " + expectedProduct);

        System.out.println("API Verification Passed");
    }

    @Test(priority = 3, dependsOnMethods = {"uiAutomationFlow", "apiVerificationFlow"})
    public void combinedValidation() {
        Response response =
                RestAssured
                        .given()
                        .contentType("application/x-www-form-urlencoded; charset=UTF-8")
                        .formParam("search_product", searchKeyword)
                        .when()
                        .post("/api/searchProduct");

        Assert.assertEquals(response.getStatusCode(), 200, "Combined validation API call failed");

        String responseBody = response.getBody().asString();

        Assert.assertEquals(uiMatchedProduct, expectedProduct, "UI matched product is incorrect");
        Assert.assertTrue(responseBody.toLowerCase().contains(uiMatchedProduct.toLowerCase()),
                "UI product not found in API response");

        System.out.println("Combined Validation Passed");
        System.out.println("UI Product  : " + uiMatchedProduct);
        System.out.println("API contains: " + uiMatchedProduct);
    }

    @AfterClass
    public void teardown() {
        if (driver != null) {
            driver.quit();
        }
    }
}
```

# 🔍 Step 6 Understand the Code Flow

## 6.1 UI Automation Flow

This method does:

* open website
* click Products
* search for `top`
* validate `Searched Products`
* capture all visible product names
* verify `Blue Top` exists in UI

Method:

```java
uiAutomationFlow()
```

---

## 6.2 API Verification Flow

This method does:

* call `POST /api/searchProduct`
* pass `search_product=top`
* validate status code `200`
* validate response is not empty
* validate `products` key exists
* validate `Blue Top` exists in response

Method:

```java
apiVerificationFlow()
```

---

## 6.3 Combined Validation

This method does:

* call API again
* compare UI matched product with expected product
* verify same product exists in API response

Method:

```java
combinedValidation()
```

# ⚠ Common Issues

## ChromeDriver issue

If Chrome does not launch:

* update browser
* update Selenium dependency
* ensure Chrome is installed

---

## Locator issue

If UI locators fail:

* inspect webpage again
* update xpath or id

---

## API mismatch issue

If API validation fails:

* check response body
* verify product spelling
* compare exact text carefully
