# Basic GitHub Upload Flow

## Step 1: Initialize Git

```bash
git init
```

## Step 2: Check Status

```bash
git status
```

## Step 3: Add Files

```bash
git add .
```

## Step 4: Commit Files

```bash
git commit -m "Initial portfolio project commit"
```

## Step 5: Rename Branch To Main

```bash
git branch -M main
```

## Step 6: Connect Remote Repository

```bash
git remote add origin <repository-url>
```

## Step 7: Push Code

```bash
git push -u origin main
```

## Step 8: Verify On GitHub

Open your GitHub repository and confirm that all required files are uploaded.
