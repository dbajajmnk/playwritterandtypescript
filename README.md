# 📘 Playwright Automation with TypeScript (Beginner → Advanced)

## 🚀 Program Overview

* **Course Title:** Playwright Automation with TypeScript
* **Duration:** 30 Hours (5 Days × 6 Hours)
* **Mode:** Instructor-led (Hands-on + Theory)

### 🎯 Target Audience

* Beginners in automation
* Manual testers transitioning to automation
* Developers (Frontend / Full Stack)

---

## 🎯 Learning Objectives

By the end of this training, participants will be able to:

* Understand web application structure (HTML, CSS, JS)
* Write TypeScript-based automation scripts
* Build end-to-end UI automation using Playwright
* Design scalable automation frameworks
* Execute tests across browsers and environments
* Integrate automation into CI/CD pipelines

---

## 📚 Course Outline

### 🧩 Module 1: Web Fundamentals (6 Hours)

#### Topics

* HTML structure & DOM
* Forms, inputs, buttons
* CSS selectors (id, class, attribute)
* Basic layouts (Flexbox)
* JavaScript fundamentals

  * Variables, functions
  * Events (click, submit)
  * DOM interaction
  * Async basics

#### 💻 Hands-on

* Build login page (HTML + CSS)
* Add validation using JavaScript

---

### 🧩 Module 2: TypeScript Essentials (4 Hours)

#### Topics

* Introduction to TypeScript
* Data types & type safety
* Interfaces & models
* Functions

  * Required parameters
  * Optional parameters
  * Default parameters
* Async/await with TypeScript

#### 💻 Hands-on

* Create reusable utility functions
* Define test data models

---

### 🧩 Module 3: Playwright Fundamentals (6 Hours)

#### Topics

* Playwright setup & project structure
* Test runner basics
* Browser, context, page concepts

#### Locators

* `getByRole`
* `getByText`
* CSS selectors

#### Actions

* click
* fill
* hover

#### Assertions

* visibility
* text validation
* state validation

#### 💻 Hands-on

* Write first Playwright test
* Automate login flow

---

### 🧩 Module 4: Advanced Playwright (8 Hours)

#### Topics

* Fixtures & hooks (`beforeEach`, `afterEach`)
* Page Object Model (POM)
* Reusable test architecture
* API testing using Playwright
* Parallel execution & configuration

#### Debugging Tools

* Trace viewer
* Screenshots
* Logs

#### 💻 Hands-on

* Convert tests into POM structure
* Add reusable fixtures
* Execute parallel tests

---

### 🧩 Module 5: Real-World Automation & Capstone (6 Hours)

#### Topics

* Authentication handling (session reuse)
* Data-driven testing (JSON/CSV)
* Cross-browser testing
* CI/CD integration (GitHub Actions / Jenkins)
* Test reporting

#### 🏁 Capstone Project

Automate complete **e-commerce flow**:

* Login
* Product search
* Add to cart
* Checkout validation

## ▶️ Getting Started

```bash
# Install dependencies
npm install

# Install Playwright browsers
npx playwright install

# Run tests
npx playwright test

# Run in headed mode
npx playwright test --headed
```
