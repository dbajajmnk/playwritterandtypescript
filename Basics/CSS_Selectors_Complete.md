# 📘 CSS Selectors (id, class, attribute) – Engineering First Template

---

# 🧠 1. WHAT

CSS Selectors are used to **target HTML elements** so we can apply styles.

Types:
- ID Selector → #id
- Class Selector → .class
- Attribute Selector → [attribute=value]

---

# ❓ 2. WHY

- To apply styles efficiently
- To target specific elements
- To avoid styling entire page unnecessarily

Without selectors:
❌ No control over UI  
❌ Poor maintainability  

---

# ⏰ 3. WHEN

| Selector | When to Use |
|----------|------------|
| ID | Unique element |
| Class | Reusable styling |
| Attribute | Conditional styling |

---

# ⚙️ 4. HOW

CSS selects elements from DOM and applies styles.

Flow:
1. HTML creates DOM
2. CSS selector targets element
3. Style applied

# 💻 6. SYNTAX

## ID
```css
#header {
  color: red;
}
```

## Class
```css
.card {
  border: 1px solid black;
}
```

## Attribute
```css
input[type="text"] {
  background: lightgray;
}
```
